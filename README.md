# 🌐 Web Design & Frontend Projects

## Project Overview

This repository features a collection of smaller web development projects created during my time at Technical High School and advanced frontend components developed during my independent programming coursework. 
Each project served as a practical opportunity to build and refine my core skills in **HTML**, **CSS**, and **JavaScript**, while learning modern web development best practices.

---

## Tech Stack & Tools

- **HTML5** – Page structure and semantic markup.
- **CSS3** – Styling, layouts, custom animations, and responsive design.
- **JavaScript (Vanilla JS)** – Interactivity, dynamic elements, DOM manipulation, and asynchronous programming.
- **Asynchronous JS** – Working with `Fetch API`, `async/await`, and processing JSON structures.
- **Media Queries** – Ensuring full responsiveness across various device screens.

---

## Featured Projects

### Async API User Directory (`/async-api-fetcher`)
A dynamic frontend component that showcases advanced data handling and UI rendering.
* **Core Features:** Asynchronously fetches user mock data from an external REST API (`JSONPlaceholder`), injects templates dynamically into the DOM, and hides the trigger action once data is loaded.
* **UI/UX Enhancements:** Implements custom CSS keyframe animations with **staggered delays** (calculated via CSS variables passed from JavaScript) to achieve a smooth cascade loading effect. Includes custom `:hover` states and click-triggered active card highlights.
